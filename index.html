CSS INTERVIEW Q/A DAY-2

1.Box-sizing:

In CSS, Box sizing property allows us to include the padding and border in an elements total width and height.Box-sizing has two different values namely content-box and border-box.By default box-sizing has set to content-box.In content-box the dimensions of the element will be actual height and width and it does not includes border and padding given to element.Border and padding takes space outside the element.If box-sizing is set to border box it includes height,width,border,padding values.but border and padding values are taken inside the element.

2.Animation Delay:

In CSS,animation delay property specifies the amount of time to wait from applying the animation to an element before starting the animation. Animation delay is specified in seconds and milli seconds.

3.To Merge cells Vertically in grid:

By using grid-row property we can merge cells vertically (Columns) in grid.There are two short hands for grid row .They are: 1.grid-row-start
2.grid-row-end

4.To Merge cells horizantally in grid:

By using grid- column property we can merge cells horizantally(rows) in grid.There are two short hands for grid column .They are: 1.grid-column-start
2.grid-column-end

5.Box Model:

CSS Box model is essentially a box around every HTML element.Box model consists of margin,border,padding,and content.

1.Margin: Margin is transperant and it is always outside the border.
2.Border: Border is around the padding and content.
3.Padding: Padding is transperant and it is present in between the border and content.
4.Content: The Content is the box where our content appears.

6.Difference between display:none; and Visibility:hidden;

display:none; -- It removes the element from the document and it does not take any space.
visibility:hidden; --It hides the element, but it takes space on the layout.
